How To Buy

$UCHIHA

See how simple it is to make a purchase using MetaMask, here you can quickly and easily learn how to buy and sell $UCHIHA

1 - Create MetaMask Wallet

Create a MetaMask Wallet using a phone or desktop computer. This will allow you to buy, sell and send $UCHIHA tokens and many other ones too.

2 - Send BNB to your wallet

You can buy Binance Coin (BNB) on centralized exchanges such as Binance. Make sure to use the BSC network when transfering BNB.

3 - Swap BNB for UCHIHA COIN

In the "To" field, click on "Select a currency", then in the "Search name or paste address" field paste the $UCHIHA Token address to find the token. The Uchiha Coin address is found below.
